# 🚨 Krisvakt Sverige
### Ett gratis tidig-varningssystem för ekonomiska och geopolitiska kriser

Krisvakt är ett personligt projekt byggt för att hjälpa vanliga människor — inte bara ekonomer — att förstå när världsläget och ekonomin förändras på ett sätt som kan påverka deras vardag.

---

## 🟢 Vad gör appen?

Appen kombinerar data från sju olika öppna källor och visar ett enkelt **trafikljus** — grönt, gult eller rött — med konkreta råd på svenska.

**Fyra flikar:**
- 📊 **Indikatorer** — VIX, räntekurva, bostadspriser, hushållsskulder och mer
- 🔮 **Framtidsrisker** — Domedagsklockan, AI & drönare, ostabila ledare, svenska valet 2026
- ⚔️ **Aktiva konflikter** — Live-data från UCDP Uppsala
- 📜 **Historiska kriser** — 2008, Eurokrisen, COVID, Ukrainakriget
